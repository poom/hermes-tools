# Resolver Cases

## Should Trigger

- "Use my-open-prs for this workflow."
- "Follow the My Open PRs procedure."
- "Troubleshoot my-open-prs behavior."

## Should Not Trigger

- "Summarize this unrelated document."
- "Run a generic code review with no mention of my-open-prs or its workflow."

These cases prove the skill resolver routes to this skill for matching requests and should not trigger for unrelated work.
