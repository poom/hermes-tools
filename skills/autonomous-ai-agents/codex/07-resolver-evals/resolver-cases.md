# Resolver Cases

## Should Trigger

- "Run codex exec to implement this feature."
- "Use Codex in full-auto mode for a long refactor."
- "Check current Codex subscription usage."

## Should Not Trigger

- "Use the GitHub app to list issues."
- "Write a Python script yourself without delegating to Codex."

These cases prove the skill resolver routes to this skill for Codex CLI operations and should not trigger for unrelated coding work.
